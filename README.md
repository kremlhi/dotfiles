Usage:

	$ git clone https://github.com/kremlhi/dotfiles.git
	$ dotfiles/install.sh -n
	$ dotfiles/install.sh

Beware! Files in the repo will overwrite files in your home directory
if they exist.

-n is dry-run, it will display what changes will be made.
