usage:

    $ git clone https://github.com/kremlhi/dotfiles.git
    $ cd dotfiles
    $ make -n
    $ make

Beware! Files in the repo may overwrite files in your home directory
if they exist.

-n is dry-run, it will display what changes will be made.
